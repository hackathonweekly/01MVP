**3.4 开发资源**



**AI Api 合集（国内外）**

```网友信息汇总，**Api 中转有风险，请自行辨别！**
```
- [硅基流动](https://siliconflow.cn/zh-cn/siliconcloud)：高性价比的 GenAI 云服务，集成了很多 API
- [openrouter.ai](https://openrouter.ai/): 比较好的 chatgpt 聚合商 
- [302.ai](https://gpt302.saaslink.net/AhFa2Z): 汇集全球顶级品牌的AI超市（包括了 api 汇总，国内外都有，chatgpt / 文生图 / 文生音乐）
- [价格](https://302.ai/pricing/)：但汇总的 api 不太全，api 价格是大概是原价格的110%
- [api2d](https://api2d.com/wiki/aiapi)：接入的不多，openai / claude / SD 
- [aimlapi.com](https://aimlapi.com/) 感觉价格偏高
- [openai-hk.com/docs/useweb.html](https://www.openai-hk.com/docs/useweb.html) 这个是宠物写真用的mj api  两毛钱一次
- [api.xi-ai.cn](https://api.xi-ai.cn/) 支持了大量的文本模型，国内外，价格还算合理
- [https://open-dev.feishu.cn/docx/P9N6dRXZYoQ3S4x2KzCcA9menhB](https://api.xi-ai.cn/https://open-dev.feishu.cn/docx/P9N6dRXZYoQ3S4x2KzCcA9menhB)
- https://2233.ai/ wild card 支持的多个模型 api
- https://cloud.google.com/vertex-ai/generative-ai/docs/partner-models/use-claude?hl=zh-cn
- 企业级Gpt中转new.xi-ai.cn 他也可以
- https://new.yunai.link/
