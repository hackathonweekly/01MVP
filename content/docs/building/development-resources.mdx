# 开发资源

本页面收集了独立开发者在开发过程中可能需要的各类API资源、云服务和开发工具，帮助你更高效地实现产品功能。

## AI API 服务

> ⚠️ **注意**：API中转服务可能存在风险，请自行评估和辨别服务提供商的可靠性。

### 聚合型API服务

- [**硅基流动**](https://siliconflow.cn/zh-cn/siliconcloud)
  - **特点**：高性价比的GenAI云服务
  - **服务**：集成多种AI模型API
  - **适用场景**：需要在国内访问AI服务的开发者

- [**OpenRouter**](https://openrouter.ai/)
  - **特点**：知名的ChatGPT聚合服务商
  - **服务**：提供多种LLM模型的统一访问
  - **适用场景**：需要灵活切换不同模型的应用

- [**302.AI**](https://gpt302.saaslink.net/AhFa2Z)
  - **特点**：汇集全球顶级品牌的AI超市
  - **服务**：ChatGPT、文生图、文生音乐等API
  - **定价**：通常是原价格的110%左右
  - **价格详情**：[302.AI定价](https://302.ai/pricing/)

### 专业API提供商

- [**API2D**](https://api2d.com/wiki/aiapi)
  - **服务**：OpenAI、Claude、Stable Diffusion等
  - **适用场景**：需要稳定访问主流AI服务的开发者

- [**Xi-AI**](https://api.xi-ai.cn/)
  - **特点**：支持大量文本模型，国内外均有
  - **优势**：价格合理，访问稳定
  - **文档**：[飞书文档](https://open-dev.feishu.cn/docx/P9N6dRXZYoQ3S4x2KzCcA9menhB)

- [**2233.AI**](https://2233.ai/)
  - **特点**：Wild Card支持的多个模型API
  - **适用场景**：需要多样化AI模型的应用

- [**云AI连接**](https://new.yunai.link/)
  - **特点**：提供多种AI模型的API接入
  - **适用场景**：国内开发者

## GPU云服务

- [**AutoDL算力云**](https://www.autodl.com/)
  - **特点**：弹性、易用、性价比高
  - **适用场景**：AI训练和推理、数据处理、科学计算

- [**RunPod**](https://www.runpod.io/)
  - **特点**：按需付费GPU云服务
  - **适用场景**：AI模型微调、大规模推理

- [**Lambda Labs**](https://lambdalabs.com/)
  - **特点**：AI研究友好的GPU云
  - **适用场景**：研究和开发AI应用

## 公共API资源

- [**Public APIs**](https://github.com/public-apis/public-apis)
  - **特点**：免费公共API的集合列表
  - **分类**：按功能领域分类的各类API
  - **适用场景**：需要集成各类功能的开发者

- [**RapidAPI**](https://rapidapi.com/)
  - **特点**：全球最大的API市场
  - **服务**：提供各种API的统一访问和管理
  - **适用场景**：需要多种API服务的项目

## 云开发平台

### 国际云开发平台

- [**Sealos**](https://sealos.io/)
  - **特点**：Kubernetes云操作系统
  - **国内版**：[sealos.run](https://sealos.run/)
  - **适用场景**：需要容器化部署的应用

### 国内云开发平台

- [**LeanCloud**](https://leancloud.cn)
  - **特点**：国内知名云服务提供商，偏向游戏云开发
  - **服务**：数据存储、用户系统、消息推送等
  - **适用场景**：移动应用后端、游戏服务器

- [**MemfireDB**](https://www.memfiredb.com/)
  - **特点**：国产Supabase替代品
  - **服务**：PostgreSQL数据库、认证、存储等
  - **适用场景**：需要在国内使用Supabase类服务的项目

- [**腾讯云开发**](https://cloudbase.net)
  - **特点**：腾讯提供的一站式后端云服务
  - **服务**：数据库、存储、云函数等
  - **适用场景**：微信生态应用、小程序后端
