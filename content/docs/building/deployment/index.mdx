# 网站部署指南

将网站成功部署到互联网是开发过程中的关键步骤。本指南提供不同技术水平用户的部署方案，从完全没有编程经验的新手到专业开发者。

## 指南分类

我们将部署指南分为两个主要部分：

### [新手部署指南](/docs/building/deployment/beginner)

适合完全没有编程经验的用户，包括：
- 一站式AI网站生成与部署工具
- 简单拖放式部署服务
- 国内外友好的托管方案指引

如果你是通过AI工具或在线生成器创建的HTML文件，或者你只是想找最简单的方式将网站发布到互联网，这个指南非常适合你。

### [进阶部署指南](/docs/building/deployment/professional)

适合有一定开发经验的专业人士，包括：
- Serverless网站部署
- 云原生PaaS平台
- 容器化部署工具
- CI/CD自动化流程
- 国内外不同地区的部署策略

如果你有一定的编程经验，需要更灵活和可扩展的部署方案，这个指南会提供更专业的工具和服务推荐。

## 选择合适的部署方案

在选择部署方案时，请考虑以下因素：

1. **技术水平**：你是否有编程经验？是否熟悉命令行和服务器管理？
2. **项目复杂度**：是简单的静态网站还是需要后端服务的复杂应用？
3. **目标受众**：你的网站主要面向国内用户还是国际用户？
4. **预算**：你是需要免费方案还是愿意投入资金获得更好的服务？
5. **扩展需求**：你是否预计网站未来会有大量流量或需要快速扩展？

根据这些因素，选择最适合你需求的部署指南开始吧！
