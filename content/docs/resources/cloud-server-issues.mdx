**Github + Vercel + 腾讯云购买域名**

- 尽量买便宜的域名
- 可以在腾讯云购买（方便万一后面要域名）
- 适用情况
- 新手入门，尝尝鲜
- 免费（访问量不大的情况下）
- 不想备案，不需要接入国内支付和短信、微信登陆，国内访问量较小
- 不适用的情况
- **需要备案+必须国内服务器**，需要接入国内支付/短信登陆，需要国内稳定访问
- Ps
- 如果你国外访问量较大，可以将 Vercel 后期切换为 **Cloudflare** 省点钱


**腾讯云轻量云服务器**

- 推荐！！ 感觉确实还可以，很省钱关键是
- 如果你不想备案，不需要接入国内支付和短信登陆，国内访问量较小，而且希望有科学上网环境，请购境外的服务器！！（ps：请勿用于非法用途，腾讯云时不时会检测）
- 如果你需要备案，需要接入国内支付/短信登陆，需要国内访问非常稳定，请购买境内服务器！
- 最低配只推荐买2核4G的，不要买比这更小的
- 价格

**腾讯云开发 CloudBase**

https://cloudbase.net/

个人版: 238.78元/年（优惠价），适用于非常小的网站，正常价格 480元/年

入门版： 1188元/年，适用于小型流量应用

超过限额的费用： https://cloud.tencent.com/document/product/876/75213

缺点

- 没有 postgresql 数据库，数据库读写有它自己的 API
- 不支持 Nextjs

优点

- 适合小程序，可以快速接入微信登陆、短信登陆等能力
- 支持云函数、云数据库、云存储等一系列服务
