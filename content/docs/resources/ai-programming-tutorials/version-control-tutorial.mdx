**代码版本控制**

**背景**

AI 经常会把代码改崩溃，有两种方式可以在 WindSurf 回退代码

**方案1：使用 Windsurf 自带的回退代码功能**
./
![](/images/version-control-tutorial.001.png)

**方案2：使用 Git （相对繁琐，但最规范）**

**如图打开 Vscode / Cursor / WindSurf 扩展商店，搜索并安装 GitLen**
./
![](/images/version-control-tutorial.002.png)

**设置你的 Git 账号**

如图，点击Windsurf右上角一个按钮打开终端
./
![](/images/version-control-tutorial.003.png)

在终端中输入以下指令（记得替换成你自己的用户名以及邮箱，一般来讲跟你的github账户一样）这样是方便让代码版本控制器 （git）知道是谁在提交内容

```Bash
git config user.name "jackie"
git con./"jackie@outlook.com"
```
![](/images/version-control-tutorial.004.png)
