*极简科技风*

- *特点：大量留白，简洁的线条，少量但精准的动效*
- *配色：以黑白为主，辅以单一突出的主题色*
- *布局：网格化布局，清晰的视觉层次*
- *适合：强调专业性和效率的产品*

*赛博朋克风*

- *特点：霓虹效果，故障艺术，不规则几何*
- *配色：深色背景，霓虹色彩（紫色、青色、粉色）*
- *布局：非对称，带有科幻感的 UI 元素*
- *适合：强调创新和前卫的产品*

*现代简约风*

- *特点：圆润的设计元素，柔和的渐变，平滑的动画*
- *配色：浅色或深色背景，渐变色彩*
- *布局：模块化，有机的形状*
- *适合：强调用户友好和现代感的产品*

*工程师风格*

- *特点：终端界面元素，代码片段，技术图表*
- *配色：深色背景，绿色/蓝色等经典编程色彩*
- *布局：类似 IDE 或终端的布局*
- *适合：强调技术和极客文化的产品*

**技巧**

1. **优化与细节调整：** 我告诉 AI 多次“继续优化界面”，结果令人惊喜！从视觉效果到动画流畅度，它真的像一个资深设计师一样，一句简单的“继续优化”或者“你觉得有哪些可以优化的点”常常比你自己不懂瞎提需求要好的多，比如它给我加了很多优秀的过渡动画，一些鼠标悬浮的小变化，更好的 UI 设计等等。
2. **一次性提多个需求**：你可以把自己当产品经理，大白话直接跟它说你的具体需求，一二三四点列出来就好，它同样也可以完成的很好（但不要提太大、复杂的需求）。
3. 你可以画好 UI，然后发给 Windsurf！

**常见错误以及解决方案**

**手动修改项目名称**

Ctrl + shift + F 全仓库搜索，查找替换 next-shadcn-template 为你的项目名称

![](/images/1h-ai-elegant-website.018.png)

**可能出现的报错以及解决方案**

**Import Error: `@/components/ui/use-toast`**

解决方案：将  @/components/ui/use-toast 替换为 @/hooks/use-toast 即可

原因：这个代码库使用到的 shadcn 最新版本的代码路径发生了变动，但是 AI 还没有更新知识库

**" can be escaped with &quot;**

，常见的另一个错误，另外 还是一堆 warning 怎么回事： 79:17  Error: " can be escaped with &quot;, &ldquo;, &#34;, &rdquo;.  react/no-unescaped-entities  79:39  Error: " can be escaped with &quot;, &ldquo;, &#34;, &rdquo;.

**Use client**

```Error:   × You're importing a component that needs useState. This React hook only works in a client component. To fix, mark the file (or its parent) with the "use client" directive.
```
在报错对应的页面上加一行下面的文本即可

```Bash
"use client"
```
