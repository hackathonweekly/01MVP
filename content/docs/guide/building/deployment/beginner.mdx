# 简易网站部署指南

这份指南专为没有编程经验的初学者设计，推荐最简单的工具和服务来部署你的MVP网站。

## 一键AI网站生成工具

### [Bolt.new](https://bolt.new) 🌍 国际友好

**特点**：
- 使用AI技术一键生成完整网站
- 所见即所得界面，无需编程
- 支持从Figma导入设计
- 集成Cursor提供代码编辑能力
- **访问注意**：国内访问可能需要良好的网络环境

### [Cursor](https://cursor.sh) 🌍 国际友好

**特点**：
- AI辅助编码工具
- 支持多种框架和语言
- 可帮助理解和修改代码
- **访问注意**：国内访问可能不稳定，需要良好的网络环境

### [Teable](https://teable.io) 🇨🇳 中国友好

**特点**：
- 中文友好的Airtable替代品
- 可用作简易网站backend
- 支持自定义视图和工作流
- **访问注意**：专为中国用户设计，国内访问速度快

## 拖拽式部署服务

无需编写代码，通过简单的拖拽操作即可部署网站。

### [Netlify Drop](https://app.netlify.com/drop) 🌍 国际友好

**特点**：
- 拖拽文件夹即可部署
- 自动分配网址
- 支持自定义域名
- **访问注意**：国内访问不稳定，部署后的网站需要全局网络环境才能流畅访问

### [GitHub Pages](https://pages.github.com) 🌏 有限访问

**特点**：
- 免费托管静态网站
- 与GitHub集成
- 简单的部署流程
- **访问注意**：国内访问可能不稳定，但部分地区可访问

### [Vercel](https://vercel.com) 🌍 国际友好

**特点**：
- 简单的部署流程
- 支持多种前端框架
- 提供免费套餐
- **访问注意**：国内访问不稳定，部署后的网站需要全局网络环境访问较快

## MVP部署选择指南

### 最简易方案（无备案需求）

如果你是完全初学者，想要在几分钟内部署一个网站：

1. **面向国际用户**:
   - Bolt.new: 使用AI一键生成网站
   - Netlify Drop: 简单拖拽部署
   - GitHub Pages: 直接上传HTML文件

2. **面向中国用户**:
   - Gitee Pages: 国内最简单的静态网站托管
   - 基础云服务+香港/新加坡节点: 无需备案，访问速度尚可
   - 临时测试: 使用Vercel等国际服务+分享测试链接

### 需要互动功能的MVP（简易数据库）

如果你需要基础的用户互动功能，可以考虑：

1. **无代码方案**:
   - Teable.io: 中国友好的Airtable替代品
   - 腾讯云静态网站+腾讯云数据库: 国内访问快速
   - Wix/Webflow等建站工具: 国际用户访问流畅

2. **低代码方案**:
   - 阿里云轻量应用服务器: 简单设置WordPress
   - 腾讯云Webify: 一键部署网站应用

## 网站更新提示

部署网站后，如何更新内容：

1. **使用Netlify/Vercel/GitHub Pages**:
   - 更新你的文件
   - 重新上传或推送到仓库
   - 自动重新部署

2. **使用Gitee Pages**:
   - 更新你的文件
   - 推送到Gitee仓库
   - 手动触发重新部署

3. **使用云存储服务（阿里云OSS/七牛云等）**:
   - 登录控制台
   - 上传新文件覆盖旧文件
   - 无需重启服务

## 针对用户群体的建议

### 完全面向国际用户
- 使用Bolt.new或Netlify Drop快速部署
- 选择国际域名（.com/.org/.io等）
- 不需要考虑备案问题

### 完全面向中国用户
- 使用Gitee Pages、腾讯云或阿里云服务
- 选择中国域名（.cn/.com.cn等）
- 如需使用自定义域名，了解[网站备案流程](/docs/building/deployment/filing)
- 考虑国内支付接口集成（如需要）

### 同时面向两类用户
- 考虑使用Cloudflare Pages（国内外访问相对稳定）
- 使用香港/新加坡节点的服务器
- 可能需要两套域名策略

## 进阶建议

- 考虑升级到更专业的部署方案
- 查看我们的[进阶部署指南](/docs/building/deployment/professional)
