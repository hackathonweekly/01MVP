**01MVP** 是周周黑客松社区旗下的产品知识库，致力于帮助创造者从0到1实现**最小可行产品**（MVP），并快速找到**前100个**用户。

## 适合谁阅读这本手册？

这本手册为以下人群设计：

- **技术爱好者**：有技术基础，想将创意变成实际产品
- **公司员工**：想在业余时间开发副业项目
- **学生创业者**：希望在校期间打造自己的产品
- **独立开发者**：寻求更高效的产品验证和开发方法
- **产品经理**：想要理解如何从0到1构建产品
- **AI探索者**：看到AI浪潮，希望打造AI相关产品

无论是出于兴趣还是商业目的，这本手册都能帮助减少弯路。

## 常见目标

读者通常有两种核心目标：

### 1. 享受创造的乐趣，解决个人问题
   
如果属于这类创造者：
- 追求的是将想法变为现实的成就感
- 希望解决日常遇到的问题
- 珍视开发过程中的学习与成长

**推荐路径**：从[入门教程](/docs/tutorials/beginners)开始，选择一个简单项目动手实践，体验从创意到实现的全过程。

### 2. 开发能产生收益的产品
   
如果有商业目标：
- 希望创造能产生收益的产品
- 需要验证市场需求，找到产品市场契合
- 追求低成本高效率的产品开发方法

**推荐路径**：从[想法验证](/docs/planning/idea-validation)开始，学习如何验证想法，然后选择合适的技术栈进行开发。

## 我们的不同

与其他创业书籍、知识库、开发者手册相比:

- **不重复造轮子，收录&连接最佳内容**: 在合适的地方链接到其他高质量的文档
- **随手可查的参考手册**: 我们鼓励在产品创造的每一步中，随时翻阅相关章节，并分享宝贵的实战经验
- **注重实操经验**: 创业理论很多书籍已经讲得很好了，我们更侧重于分享实践中的具体经验
- **从小案例到大系统**: 提供从简单到复杂的实际案例，帮助循序渐进建立信心

## 最好的时代

这是一个最好的时代，能做的事情远远超出了想象。利用现代工具和AI技术，一个人就能完成产品开发，这在以前是无法想象的。

AI技术至少降低了:

- **学习的门槛**: ChatGPT等AI助手大大降低了技术学习的难度
- **开发的门槛**: 可以利用AI技术，一人之力，完成一个产品的开发
- **验证的成本**: 通过AI工具快速生成原型，以低成本测试产品理念

## 从何开始？

根据经验和目标，我们推荐不同的起点：

### 完全新手

1. 先阅读[开发环境设置](/docs/introduction/setup-and-tools)
2. 尝试完成一个[入门级教程](/docs/tutorials/beginners)
3. 学习[MVP规划基础](/docs/mvp-planning)
4. 查看[MCP教程](/docs/mcp-tutorials)了解实用开发方法

### 有编程基础

1. 选择一个[进阶教程](/docs/tutorials/intermediate)上手实践
2. 了解[产品验证方法](/docs/planning/idea-validation)
3. 探索[AI工具使用指南](/docs/introduction/ai-tools)
4. 参考[MCP教程](/docs/mcp-tutorials)加速产品开发

### 已有产品想法

1. 学习[想法验证技巧](/docs/planning/idea-validation)
2. 了解[用户研究方法](/docs/planning/user-research)
3. 查看[MVP范围定义](/docs/mvp-planning/defining-scope)
4. 通过[MCP教程](/docs/mcp-tutorials)快速实现产品原型
