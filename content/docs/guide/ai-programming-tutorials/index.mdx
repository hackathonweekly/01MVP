## 入门教程

### [10分钟克隆猫灯网站入门教程](/docs/resources/ai-programming-tutorials/getting-started-10min-cat-lamp-website-clone)

### [生日祝福网站示例](/docs/resources/ai-programming-tutorials/birthday-wishes-website-example)
学习如何创建一个个性化的生日祝福网站，这是一个简单但完整的Web应用示例。

### [创建Match3游戏及更多示例](/docs/resources/ai-programming-tutorials/creating-a-match3-game-plus-more-examples)
探索如何使用AI工具快速开发一个简单的Match3游戏，以及其他游戏开发示例。

## 学习路径建议

如果你是编程新手，我们建议按以下顺序学习：

1. 首先完成[10分钟克隆猫灯网站入门教程](/docs/resources/ai-programming-tutorials/getting-started-10min-cat-lamp-website-clone)
2. 学习[安装Node.js和npm](/docs/resources/ai-programming-tutorials/installing-nodejs-npm)和[Cursor安装和使用](/docs/resources/ai-programming-tutorials/cursor-installation-and-usage)
3. 尝试[生日祝福网站示例](/docs/resources/ai-programming-tutorials/birthday-wishes-website-example)
4. 学习[版本控制教程](/docs/resources/ai-programming-tutorials/version-control-tutorial)
5. 根据兴趣选择其他教程深入学习

对于有经验的开发者，可以直接选择感兴趣的主题学习，或者探索如何将AI工具集成到现有的开发流程中。
