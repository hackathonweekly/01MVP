## 项目指南结构

每个项目指南包含以下内容：

- **项目概述**：描述项目目标和功能
- **操作步骤**：详细的实施流程
- **AI提示词**：用于向AI请求帮助的提示词示例
- **关键要点**：项目中的重要技术和设计决策
- **成果展示**：最终项目的效果和功能

## 如何使用这些项目指南

1. 选择感兴趣的项目，阅读项目概述和操作步骤
2. 准备必要的环境和工具
3. 按照步骤实施，使用提供的AI提示词向AI助手（如Claude、ChatGPT）请求帮助
4. 尝试根据自己的需求进行自定义和扩展

## 学前准备

开始这些项目前，建议你准备：

1. 一个代码编辑器（推荐VS Code或Cursor）
2. 现代浏览器（Chrome或Firefox）
3. AI助手账号（如Claude、ChatGPT）
4. 基础的编程知识（视项目而定）
5. 必要的API账号（如涉及特定API调用）
