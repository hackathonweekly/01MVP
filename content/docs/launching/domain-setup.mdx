**购买域名**

**Tl;dr**

**cloudflare**

核心劣势：无法自定义 dns / 60 天内不可转出 / 便宜的域名在上面买不到  / 普通域名续费还行

价格表在这里：https://cfdomainpricing.com/

除了便宜之外，Cloudfare 购买域名还有以下优势：

**优势：**

1. **安全性保障**
1. **内置 DNSSEC**：Cloudflare 提供内置的 DNSSEC（域名系统安全扩展），有效防止域名劫持、**DNS 缓存投毒等攻击，确保域名解析安全。**
2. **DDoS 防护**：凭借其**强大的 DDoS 防护能力**，Cloudflare 可有效抵御大规模攻击，确保域名和网站在遭遇恶意流量时保持稳定。
3. **SSL/TLS 支持**：Cloudflare **提供免费的 SSL/TLS 证书**，轻松启用 HTTPS 加密传输，提升用户隐私和数据安全。
4. **WHOIS 隐私保护**：Cloudflare 自动为注册的域名启用 WHOIS 隐私保护，免费隐藏域名所有者的个人信息，避免垃圾邮件和潜在的隐私泄露。
5. **零加价政策**：Cloudflare 承诺域名注册价格仅包括注册局费用，**无额外加价或隐藏费用**，确保用户支付的价格透明公正。
6. **统一管理面板**：通过 Cloudflare，用户可以在一个平台上轻松管理域名、DNS、CDN、SSL 证书等，简化管理流程，提升效率。
7. **无广告政策**：与部分域名注册商不同，Cloudflare 不会在你的网站或管理界面上投放广告，提供干净的用户体验。
2. **性能提升**
1. **快速 DNS 解析**：Cloudflare 以其全球领先的 DNS 解析速度著称，**确保用户可以更快访问你的网站，提升网站体验。**
2. **全球 CDN 支持**：Cloudflare 的全球内容分发网络（CDN）通过将网站内容分发至全球节点，显著加速跨地域访问速度，**提升网站加载效率**。

**劣势：**

**自定义 Nameservers 限制**：Cloudflare 免费版本不支持自定义 Nameservers，这意味着你无法使用自定义的域名服务器，而只能**通过添加 A 记录来指向服务器/空间的 IP 地址**。

**不支持所有顶级域名（TLDs）**

- **TLDs 选择有限**：Cloudflare 并未支持所有可用的顶级域名（TLDs），**部分冷门或国家代码域名（如 .cn、.tw）可能无法通过 Cloudflare 注册。**

**需要较强的技术基础**

- **初学者学习曲线较高**：尽管 Cloudflare 提供强大的 DNS、SSL、CDN 和安全功能，但对于**不熟悉网络和网站管理的初学者来说，理解和配置这些功能可能有一定的难度。**
