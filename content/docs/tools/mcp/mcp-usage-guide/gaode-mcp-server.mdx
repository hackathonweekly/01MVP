# 高德地图MCP服务集成指南

本教程将指导您如何配置并使用高德地图MCP服务，该服务可让Claude通过MCP协议实现地点搜索、旅游路线规划、周边设施查询等功能，为旅游规划提供更加可视化与合理的支持。

## 准备工作

在开始前，您需要：

1. 一个高德开放平台API密钥
2. Node.js环境
3. 已配置好的MCP环境（如Cursor编辑器）

## 获取高德地图API密钥

1. 访问[高德开放平台](https://lbs.amap.com/)
2. 注册并登录账号
3. 在控制台创建应用，选择Web服务API
4. 获取API Key（请妥善保管，不要泄露）

## 安装高德地图MCP服务

您可以选择以下两种方式之一：

### 方式一：使用NPM安装（推荐）

1. 安装服务：
```bash
npm install @wopal/mcp-gaode-maps
```

2. 在MCP配置文件中添加：
```json
{
  "mcpServers": {
    "gaode": {
      "command": "cmd",
      "args": [
        "/c",
        "npx",
        "@wopal/mcp-gaode-maps"
      ],
      "env": {
        "GAODE_API_KEY": "您的高德地图API密钥"
      }
    }
  }
}
```

### 方式二：从源码安装

1. 克隆或下载代码：
```bash
git clone https://github.com/wopal-dev/mcp-gaode-maps.git
```

2. 安装依赖：
```bash
cd mcp-gaode-maps
npm install
```

3. 在MCP配置文件中添加：
```json
{
  "mcpServers": {
    "gaode": {
      "command": "cmd",
      "args": [
        "/c",
        "npx",
        "ts-node",
        "路径/到项目/src/index.ts"
      ],
      "env": {
        "GAODE_API_KEY": "您的高德地图API密钥"
      }
    }
  }
}
```

## 功能与使用示例

高德地图MCP服务提供以下核心功能：

### 1. 智能旅游路线规划

最多支持16个途经点的旅游路线规划，自动计算最优顺序，并提供可视化地图链接。

**使用示例**：
```
请帮我规划一条上海三日游路线，包括外滩、东方明珠、迪士尼、豫园、南京路，并提供可视化地图
```

### 2. 景点搜索与详情查询

查询景点的详细信息，包括评分、开放时间、门票价格等。

**使用示例**：
```
请查询黄山风景区的开放时间、门票价格和旅游季节推荐
```

### 3. 周边设施查询

根据景点位置查询周边的餐厅、酒店、停车场等设施。

**使用示例**：
```
请推荐西湖附近评分最高的五家杭帮菜餐厅
```

### 4. 天气查询

查询目的地的天气情况，为旅游规划提供参考。

**使用示例**：
```
请查询三亚未来三天的天气情况，并给出适合的旅游活动建议
```

## 高级使用技巧

### 多景点最优路线

当需要规划多个景点的参观顺序时，可以使用以下提示：

```
请帮我规划一条最省时的路线，依次游览北京的故宫、天坛、颐和园、长城和鸟巢，考虑交通时间和各景点的游览时长
```

服务将自动计算最优路线顺序，并提供可视化地图链接。

### 结合天气的智能推荐

结合天气查询与景点推荐：

```
请根据杭州明天的天气情况，推荐适合的西湖游览路线和参观景点
```

### 特色餐厅与住宿推荐

基于位置的餐厅和酒店推荐：

```
请推荐在西安古城墙附近，既有当地特色又评分高的餐厅和精品酒店
```
